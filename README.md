# tic-tac-toe
Project 1: Tic Tac Toe

Implement a GUI Tic Tac Toe Game

## Deliverables / Requirements
- [ ] JavaFX GUI board (3x3 buttons)
- [ ] Player name entry before game starts
- [ ] Win + tie detection
- [ ] +100 points per win
- [ ] Top 5 scoreboard display
- [ ] Save/load scoreboard (optional)
